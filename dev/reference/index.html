KnetNLPModels.KnetNLPModel — Type

KnetNLPModel

Data structure that interfaces neural networks defined with Knet.jl as an NLPModel.

KnetNLPModels.build_array — Method
build_array(vec, var_layer, index)

Inverse of the function Knet.cat1d, it generates a KnetArray similar to varlayer from the vector vec. The values are those of the vector in the range of index to index+consumedindex. This method is not optimised, it consumes memory.

source
KnetNLPModels.build_nested_array_from_vec — Method
build_nested_array_from_vec(model, v)

Build a vector of KnetArrays from v similar to Knet.params(model.chain). Call build_array iteratively to build each intermediary KnetArrays. This methods is not optimised, it consumes memory.

source
KnetNLPModels.build_nested_array_from_vec — Method
build_nested_array_from_vec(chain, v)

Builds a vector of KnetArrays of the same shape as chain of value v. It calls iteratively build_array to build each intermediary KnetArrays. This method is not optimised, it consumes memory.
